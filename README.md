# Underthink

The science of shower thoughts, walking, and letting your mind work.

**https://underthink.app**

A Jekyll site hosted on GitHub Pages. Home of the Walk and Clear Android apps.

## Local development

```bash
bundle install
bundle exec jekyll serve
```

## Editing content

All content pages are markdown files in the root directory. Edit them directly on GitHub or locally, commit, and push. GitHub Pages builds automatically.

## Structure

- `_layouts/` — HTML templates
- `assets/css/` — Stylesheet
- `_config.yml` — Jekyll configuration
- `*.md` — Content pages
